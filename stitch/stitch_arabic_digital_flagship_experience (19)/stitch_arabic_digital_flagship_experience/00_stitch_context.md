# Tashtep UI Context

## Project

Tashtep is a premium Egyptian digital flagship store for paints,
decorative materials, wallpapers and finishing supplies.

## Design Philosophy

50% Zara Home

30% Apple

20% IKEA

## Language

Arabic

Native RTL

## Design Principles

- Luxury
- Editorial
- Minimal
- Spacious
- Premium photography
- Soft shadows
- Large whitespace

## Never

- Never redesign Header
- Never redesign Footer
- Never redesign Product Card
- Never redesign Search
- Never change typography
- Never change colors
- Never create generic ecommerce templates

## Always

Reuse existing components.

Desktop first.

Responsive.

Accessibility first.

Performance first.

Maintain visual consistency across every page.

Create a digital flagship experience instead of a traditional online store.
