# Tashtep Enterprise Platform

Tashtep is a premium ecommerce platform in Egypt specializing in high-end paints, decorative materials, wallpapers, tools, and finishing supplies.

## Documentation
Please refer to the `/docs` folder for architectural references:
- PRD

## Architecture
- Next.js 15 App Router
- React 19 (Server Components First)
- Tailwind CSS & shadcn/ui
- Prisma ORM & MySQL
- Auth.js
